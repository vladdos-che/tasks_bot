# Инструкция по запуску проекта

Проект состоит из трех основных компонентов: Telegram-бота, бэкенда (FastAPI) и фронтенда (React + Vite).

## Подготовка окружения (Бот и Бэкенд)
Боту и бэкенду требуются общие Python-зависимости.

1. Убедитесь, что у вас установлен Python (рекомендуется 3.9+).
2. Создайте виртуальное окружение в корне проекта:
   ```bash
   python -m venv .venv
   ```
3. **Активируйте виртуальное окружение** (это важный шаг, без него команды работать не будут!):
    * Для **Linux/macOS** обязательно используйте команду `source`:
      ```bash
      source .venv/bin/activate
      ```
    * Для **Windows**:
      ```bash
      .venv\Scripts\activate
      ```
4. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
5. Убедитесь, что в корне проекта существует настроенный файл `.env` (в нём должен быть как минимум `BOT_TOKEN`).

---

## 1. Запуск Бота
Бот использует `aiogram` и запускается через `main.py` в корне проекта.

**Команда для запуска** (из корня проекта, при активированном виртуальном окружении):
```bash
python main.py
```
*Бот начнет принимать сообщения (polling) и поднимет внутренний webhook-сервер на порту 8001.*

---

## 2. Запуск Бэкенда (API)
Бэкенд написан на FastAPI и запускается через `uvicorn`. Файлы API находятся в папке `api/`.

**Команда для запуска** (из корня проекта, при активированном виртуальном окружении):
```bash
uvicorn api.main:app --reload --port 5252
```
*API будет доступно по адресу http://localhost:5252. Документация (Swagger) доступна по адресу http://localhost:5252/docs.*

---

## 3. Запуск Фронтенда
Фронтенд находится в папке `frontend/`. Для его работы потребуется установленный [Node.js](https://nodejs.org/).

1. Перейдите в директорию фронтенда:
   ```bash
   cd frontend
   ```
2. Установите зависимости (если они еще не установлены):
   ```bash
   npm install
   ```
3. Запустите сервер для разработки (Vite):
   ```bash
   npm run dev
   ```
*Фронтенд будет доступен по адресу, который выдаст Vite (обычно это http://localhost:5173).*
